# Strip Club Map Assets List

This document outlines the required custom textures and models needed for implementing the strip club themed multiplayer map in ET Legacy.

## Required Models

### Interior Elements
- **Dancer poles**
  - Central stage poles (x3)
  - VIP area poles (x2)
  - Appropriate animations and reflective properties

- **Bar with bottles**
  - Main bar counter with detailed surface
  - Bar stools (x8)
  - Liquor bottles and glasses on shelves
  - Cash register

- **DJ booth with equipment**
  - DJ table with mixers and equipment
  - Speakers (large x2, small x4)
  - Lighting control panel

- **Lounge seating**
  - VIP couches (x4)
  - Private booth seating (x6)
  - Standard tables and chairs (x10 sets)

- **Safe for the office**
  - Large wall safe with detailed door
  - Destructible components
  - Interior money/valuables visible when opened

### Structural Elements
- **Doors**
  - Main entrance double doors
  - Staff/backstage door
  - VIP entrance door
  - Office door
  - Side entrance (constructible)

- **Stage elements**
  - Main stage platform with steps
  - Stage lighting rigs
  - Curtains and backdrop

- **Office furniture**
  - Desk with computer
  - Office chair
  - Filing cabinets
  - Wall decorations

## Implementation Notes

1. All textures should be optimized for performance while maintaining the adult-themed atmosphere
2. Models should have appropriate LOD (Level of Detail) settings for performance
3. Consider reusing and modifying existing assets where possible
4. Ensure all assets are properly UV mapped and textured
5. Maintain consistent scale across all models
6. Include appropriate collision meshes for all interactive elements
